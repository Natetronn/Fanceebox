Fanceebox

Fanceebox is a add-on for ExpressionEngine 2.x. It's main purpose is to allow for fancybox.net style lightbox popups in the control panel. You can then create popups for your custom fields' labels. This is a nice feature for settings and instructions so your clients have them right where they need them.

Fanceebox

Fanceebox is a very basic accessory though, very useful. What it does is add fancybox to the control panel so you can call it from your custom fields’ labels.

Read this article for a bit more context on adding HTML to your field labels.

You can add popups links to images, galleries and websites directly within your publish/edit page. These can be external or internal.

New
Now with fancybox support for your custom CP navigation via Zoo Flexible Admin.

Let me know if you need any assistance and or can contribute to this add-on. Contact me on twitter or via my website.

Compatibility

  • ExpressionEngine 2.x+
  • Tested in 2.1.3, 2.2.0, 2.2.1, 2.2.2, 2.3.1

Installation

For EE2

  • Copy system/expressionengine/third_party/fanceebox to your system/expressionengine/third_party directory
  • Copy themes/third_party/fanceebox to your themes/third_party directory

Usage

You can add one of the following classes to your a href tag in your custom field description:

  • fancy_image
  • fancy_group
  • fancy_inline
  • fancy_iframe

Examples

Single item

<a class="fancy_iframe" href="http://fancybox.net">Fancybox</a>

Gallery of items

<a class="fancy_group" rel="group1" href="image1.jpg">Step 1</a>
<a class="fancy_group" rel="group1" href="image2.jpg">Step 2</a>

Zoo Flexible Admin

Add ?iframe to the end of your custom links via Zoo Flexible Admin’s control panel.

Notes

You can update popup box sizes and settings however, at this time these options will need to be updated manually in the system/expressionengine/third_party/fanceebox/javascript/jquery.fanceebox.js file to meet your specific needs.
Please see the fancybox api and their howto for more options.
